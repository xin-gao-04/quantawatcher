# M2.2 可靠性与提示词增强

## 范围
- 晨报刷新链路拆分为快照优先 + 研究数据异步，避免慢源阻塞 UI。
- 数据源超时/重试与降级策略优化（AkShare/Tencent）。
- Web UI 优化：状态可视化、行情速览、异常/超时提示。
- 盘后提示词增强：证据结构化、置信度校准、数据覆盖评分。

## 默认值
- AkShare 快照超时：12s；研究超时：6s
- Tencent 超时：8s；重试 2 次
- 研究数据开关：开启；最大标的数：8

## 配置项
- QW_AKSHARE_SNAPSHOT_TIMEOUT_SEC
- QW_AKSHARE_RESEARCH_TIMEOUT_SEC
- QW_TENCENT_TIMEOUT_SEC / QW_TENCENT_RETRIES / QW_TENCENT_BACKOFF_SEC
- QW_RESEARCH_ENABLED / QW_RESEARCH_MAX_SYMBOLS

## 验收标准
- 刷新晨报后 5 秒内能看到状态更新与已生成的晨报内容。
- 即使研究数据超时或失败，晨报与行情榜单仍可刷新展示。
- UI 能展示数据来源与降级状态，错误提示明确。
- 盘后提示词输出严格 JSON，包含 evidence 与 data_quality_score 字段。
