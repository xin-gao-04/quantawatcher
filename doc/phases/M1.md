# M1 最小骨架落地（v0.1）

目标：搭建可运行的服务骨架，支撑后续数据源/指标/规则的渐进式接入。

## 已落地范围
- FastAPI 服务与基础健康检查
- APScheduler 任务调度（快照轮询 + 晨报）
- SQLite 存储（任务运行记录 + 事件表）
- 晨报 Markdown 生成 + 发送（WeCom）
- 插件接口草案（Collector/Indicator/Strategy/Notifier/Report）

## 关键路径
- 启动服务：`python -m uvicorn app.api.main:app --reload`
- 健康检查：`GET /health`
- 晨报任务：固定时刻触发并推送（若 webhook 未配置，记录告警日志）

## 出口（可验证）
- 服务可启动并保持运行
- `task_runs` 能记录任务执行结果
- `doc/tech/tech_log.md` 记录本阶段技术积累

## 测试用例（最小集）
- SQLite 初始化与写入：`tests/test_storage_sqlite.py`
- 晨报格式：`tests/test_morning_brief.py`
- 健康检查：`tests/test_api_health.py`
