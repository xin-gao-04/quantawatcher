# M1.4 盘后复盘闭环 + 报告参数（v0.1）

目标：补齐盘后复盘结构化数据与报告输出，支持 UI 调参与自动推送，并将晨报/盘后共用的指标层落地。

## 范围
- 盘后复盘结构化数据生成与存档
- 盘后复盘 Markdown 报告生成与推送
- 报告参数（动量窗口、异常阈值、强弱阈值、盘后时间）可通过 API/前端修改
- watchlist 历史快照缓存，用于动量指标计算
- UI 增加盘后复盘预览与参数设置

## 默认值
- 盘后报告时间：15:30
- 盘后数据刷新：15:10
- 动量窗口：5/20 日
- 异常阈值：3%
- 强弱阈值：2%

## 新增接口
- `POST /reports/post-close/refresh`
- `GET /reports/post-close`
- `GET /reports/post-close/data`
- `POST /reports/post-close/send`
- `GET /reports/post-close/refresh/status`
- `GET /reports/params`
- `POST /reports/params`

## 验收出口（可验证）
- 盘后刷新完成后，`GET /reports/post-close` 返回完整 Markdown
- 盘后数据成功落地至 `data/post_close_report_data.json`
- 报告参数修改后，盘后/晨报指标输出发生变化
- UI 可手动刷新并发送盘后复盘
