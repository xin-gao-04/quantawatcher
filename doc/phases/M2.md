# M2 A股行情接入（AkShare）

目标：用最短路径接入 A 股现货行情，为晨报提供真实数据。

## 新增能力
- AkShare 现货快照：watchlist + 全市场榜单
- 晨报刷新接口：`POST /reports/morning-brief/refresh`
- watchlist JSON：`data/watchlist.json`
 - 晨报刷新调度：`QW_MORNING_BRIEF_REFRESH_HOUR/MINUTE`

## 出口（可验证）
- `refresh` 后 `data/morning_brief_data.json` 被更新
- `GET /reports/morning-brief` 输出包含榜单与 watchlist 信息
