# M1.2 晨报结构化数据（可编辑/可预览）

目标：晨报不仅能手工编辑，还可以维护结构化数据并自动生成模板内容。

## 新增能力
- 晨报结构化数据文件：`data/morning_brief_data.json`
- API 管理结构化数据：`/reports/morning-brief/data`
- 晨报生成逻辑：优先使用草稿；无草稿时基于结构化数据生成模板

## 出口（可验证）
- `GET /reports/morning-brief/data` 返回数据
- `POST /reports/morning-brief/data` 保存后立即生效
- 不提供草稿时，`GET /reports/morning-brief` 使用结构化数据生成
