# M2.3 Eastmoney 直连补全与网络诊断

## 范围
- 新增 Eastmoney 直连补全源（分页 + 限速）。
- AkShare 失败时尝试 Eastmoney 直连，再降级到腾讯行情。
- 网络诊断脚本用于验证代理与 eastmoney 域名可达性。
- Sina 全市场补全源用于 eastmoney 不可达场景。

## 默认值
- Eastmoney host 列表：`82.push2.eastmoney.com,push2.eastmoney.com`
- 分页大小：200
- 最大页数：50
- 页间延迟：0.2s

## 配置项

- QW_EASTMONEY_HOSTS
- QW_EASTMONEY_TIMEOUT_SEC
- QW_EASTMONEY_PAGE_SIZE

- QW_EASTMONEY_PAGE_DELAY_SEC
- QW_EASTMONEY_USER_AGENT
- QW_EASTMONEY_COOKIE
- QW_EASTMONEY_AUTO_COOKIE

- QW_SINA_MARKET_PAGE_SIZE
- QW_SINA_MARKET_MAX_PAGES
- QW_SINA_MARKET_PAGE_DELAY_SEC

- QW_SINA_MARKET_SORT

## 验收标准
- eastmoney 可达时，晨报榜单不依赖 AkShare。
- eastmoney 不可达时，诊断脚本能明确定位问题。 
- sina 可达时，榜单仍可补齐。
